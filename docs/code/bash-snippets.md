# Utilities

zip and unzip files
- `-c` create an archive
- `-z` compress the archive with gzip
- `-x`
- `-v` verbose mode with progress
- `-f` filename

```bash
# zip
tar -czvf name-of-archive.tar.gz /path/to/directory-or-file
# extract
tar -zxf name-of-archive.tar.gz /path/to/directory-or-file
```
